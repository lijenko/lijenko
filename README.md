## 🚀 Projetos em Destaque

> Todos os repositórios são privados pois envolvem projetos reais de clientes. Abaixo um resumo do que foi desenvolvido:

---

### 🥽 MTechVR — Plataforma de Realidade Imersiva
**Stack:** React 19 · TypeScript · Three.js · Framer Motion · Tailwind CSS · Supabase

Plataforma web completa para apresentação de serviços de **Realidade Virtual, Aumentada e Mista**.
- Visualizações 3D interativas com **React Three Fiber**
- Internacionalização (PT/EN)
- Hero com video background e animações suaves
- 8 páginas dinâmicas de serviço
- 🌐 [mtechvr.com.br](https://mtechvr.com.br)

---

### 📊 Extrator Fiscal — Plataforma EFD/SPED
**Stack:** TypeScript · React · Node.js · TRPC · Prisma ORM · Vite · Mercado Pago API

Plataforma **full stack** para extração e análise de dados fiscais (EFD/SPED).
- API backend com **TRPC** e **Express**
- Integração com **Mercado Pago** para pagamentos
- Arquitetura monorepo (client / server / shared)
- Deploy automatizado no **Vercel**
- 🌐 [extrator-fiscal.com.br](https://extrator-fiscal.com.br)

---

### 🏛️ WayTax Consulting — Site de Consultoria Fiscal
**Stack:** React 19 · TypeScript · Tailwind CSS 4 · Radix UI · Zod · Wouter

Site institucional moderno para empresa de **consultoria fiscal e compliance**.
- Formulários com validação via **React Hook Form + Zod**
- Roteamento com **Wouter** e páginas de serviços dinâmicas
- Componentes acessíveis com **Radix UI**
- 🌐 [waytax.com.br](https://waytax.com.br)

---

### 🤖 Exchange Group Bot — Bot para grupos de exchange
**Stack:** JavaScript · Node.js · SQLite · PM2

Bot de automação para **gerenciamento de grupos de uma exchange**, com persistência local e deploy 24/7 em VPS.
- Processamento de comandos em tempo real via messaging API
- Persistência de dados com **SQLite** e exportação CSV
- Gerenciamento dinâmico de grupos sem reinicialização
- Deploy contínuo em **VPS** com **PM2**

---

### 🧾 Ecredac — Plataforma Web
Site institucional moderno para consultoria tributária e crédito acumulado de ICMS.

- Frontend em React 19 + TypeScript + Vite + Tailwind CSS 4
- Roteamento SPA com Wouter e páginas dedicadas para serviços, FAQ, contato e política de privacidade
- Formulário de contato com máscaras, validação manual, captura automática de CNPJ e envio de leads via API/Resend
- UI utilitária com componentes estilo shadcn (Radix Slot + class-variance-authority)
- SEO e performance com metadados centralizados, Open Graph, JSON-LD, Google Analytics/Ads e consentimento LGPD
- 🌐 [ecredac.com.br](https://ecredac.com.br)
---

### 🧑‍🔧 Autônomo — Plataforma para Autônomos
**Stack:** HTML · CSS · JavaScript

**Projeto Acadêmico** — Site focado em soluções para profissionais autônomos.
- 🌐 Demo: [autonomo-drab.vercel.app](https://autonomo-drab.vercel.app)

---

### 🏦 BankSecure — Sistema de Gerenciamento de Apólices
**Stack:** Java 21 · Spring Boot · Spring Data JPA · H2 Database · JUnit 5 · Mockito · JaCoCo · Docker

**Projeto Acadêmico em grupo** (FIAP) — Sistema completo de gerenciamento de apólices e propostas de seguros.
- API REST com **Spring Boot** e arquitetura em camadas (Controller / Service / Repository)
- Cobertura de testes com **JUnit 5**, **Mockito** e relatórios via **JaCoCo**
- Autenticação de funcionários e regras de elegibilidade de clientes
- Dashboard com receita total e receita por tipo de seguro
- Containerizado com **Docker**
- 🔗 [Ver repositório](https://github.com/raquelrleite/banksecure)
